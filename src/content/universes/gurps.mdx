## Setting

GURPS, first published in 1986, is deliberately setting-agnostic. Rather than defining a world, it provides a point-buy character creation system and modular rules that can be assembled to simulate any genre from historical to science fiction to fantasy. GURPS Magic (first published as a supplement in 1989) provides the default spell system — a skill-based approach where each spell is learned as an individual skill with prerequisites forming a tree structure. Later supplements introduced alternative frameworks: Thaumatology expanded magical theory, Ritual Path Magic offered a freeform approach, and various setting books customized magic for specific worlds like Banestorm's Yrth.
