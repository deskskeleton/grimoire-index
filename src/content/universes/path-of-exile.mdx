## Setting

Wraeclast is the setting of Path of Exile (2013) and Path of Exile 2. The continent is a place of exile — a ruined land of ancient civilizations, eldritch corruption, and desperate survival. The Vaal civilization's experiments with Virtue Gems created a system where magical abilities are externalized into physical gems that can be socketed into equipment. Skill gems grant active abilities, while support gems modify those abilities (adding chain, multiple projectiles, elemental conversion, etc.). This creates a combinatorial magic system where the same base spell can function completely differently depending on how it is supported and linked, making character building an exercise in systems engineering.
